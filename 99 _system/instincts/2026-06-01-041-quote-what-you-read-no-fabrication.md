# Quote what you read, no fabrication

When citing a file, use file_path:line_number. When citing a fact, cite the source. No fabricated file paths, IDs, or quotes. If a quote is uncertain, say so explicitly. If a file doesn't exist on disk, don't pretend it does.

When citing any file, ID, or quote
