# Decisions Log

> Records what was decided, by whom, and on what evidence. The Researcher and Mavis both write here.

## Format

```yaml
- id: dec-YYYY-MM-DD-NNN
  decision: "<one sentence>"
  decided_by: <researcher | mavis | hermes | andre>
  reason: "<one sentence>"
  evidence:
    - src-...
    - claim-...
  supersedes: dec-...   # if it overturns an earlier decision
  date: YYYY-MM-DD
```

## Recent decisions

*Empty. Append as decisions are made.*

---

**Discipline:** No silent changes. If the source plan shifts, the dossier set changes, or a guardrail relaxes, it goes here. The vault without a decision log is a vault that forgets why it changed direction.
